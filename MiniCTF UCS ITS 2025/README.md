# MiniCTF UCS ITS 2025

Seleksi teknikal [Unit Cyber Security ITS](https://www.instagram.com/ucs_its/), berupa kompetisi CTF dengan sistem _jeopardy_.

Ada 7 kategori, dengan total 16 soal:

- Beginner
- Digital Forensic
- Web Exploitation
- Binary Exploitation
- Cryptography
- Reverse Engineering
- Blockchain
